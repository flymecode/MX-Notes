②  apt-get常用命令

| 命令             | 描述                                   |
| ---------------- | -------------------------------------- |
| apt-get  install | 下载并安装软件包                       |
| apt-get  upgrade | 下载并安装本系统上已有的软件包的最新版 |
| apt-get  remove  | 卸载特定的软件包                       |
| apt-get source   | 下载特定软件的源代码                   |
| apt-get clean    | 删除所有的已下载的包文件               |
